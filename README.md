# 🚀 SivaGPT – AI Powered Full Stack Application

## 📌 Overview

SivaGPT is a full-stack web application that integrates **Generative AI** with a **Spring Boot backend** and a **React frontend**. The application allows users to send messages and receive intelligent AI-generated responses in real time.

---

## 🛠️ Tech Stack

### Backend

* Java
* Spring Boot
* Spring AI (ChatClient)
* REST APIs

### Frontend

* React.js
* HTML
* CSS
* JavaScript

### Tools

* Git & GitHub

---

## ⚙️ Features

* 🤖 AI-powered chat responses
* 🔗 REST API endpoint (`/api/{message}`)
* 🌐 Frontend-backend integration
* 🔓 Cross-Origin support (CORS enabled)
* ⚡ Real-time response handling

## 🚀 Getting Started

### 🔧 Backend Setup
1. Navigate to backend folder:
   ```bash
   cd backend
````

2. Run the Spring Boot application:

   ```bash
   mvn spring-boot:run
   ```

---

### 💻 Frontend Setup

1. Navigate to frontend folder:

   ```bash
   cd frontend
   ```
2. Install dependencies:

   ```bash
   npm install
   ```
3. Start the application:

   ```bash
   npm start
   ```

---

## 🔌 API Usage

### Endpoint:

```
GET /api/{message}

### Example:

```
